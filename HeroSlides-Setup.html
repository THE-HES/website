HeroSlides Tab Setup

Download a pre-filled CSV and import it into your Google Sheet as the HeroSlides tab — takes about 30 seconds.

  1. Click Download HeroSlides CSV below.

  2. Open your Google Sheet (the H.E.S. master sheet).

  3. Click the + button at the bottom to add a new tab. Name it exactly HeroSlides.

  4. With the new tab selected, go to File → Import, click Upload, and drag the downloaded CSV onto the upload area.

  5. In the import dialog: set Import location to Replace current sheet and Separator type to Comma. Click Import data.

  6. Done! Your slides are live. Edit any row, add new ones, or set an Expiry Date using the date picker in column J.

✅ CSV downloaded! Follow the steps above to import it.
Column guide:
Active = yes / no — toggle a slide on/off instantly
Theme = use | to split lines: "Summer Camp | 2026 is | now open." — line 2 is always yellow
Expiry Date = click any cell in column J and use Google Sheets' date picker — the slide auto-hides the day after
Image URL = leave blank to use the built-in H.E.S. photos for Camp / Fair slides
