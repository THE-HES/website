2Step-by-Step Import Instructions
⚠ Do this for each of the 4 tabs — one at a time Don't try to import all 4 at once. Open the correct tab in Google Sheets, then follow the steps below for each file.
  1. Open the H.E.S. Master Google Sheet
    Go to sheets.google.com and open the H.E.S. Brain spreadsheet. You'll see tabs at the bottom: Org · HeroSlides · Website · Programs · Events · Calendar · SubPrograms.
  2. Click on the tab you want to update (e.g. "Org")
    Start with Org, then do Programs, then SubPrograms, then Events.
  3. Select all existing data in the tab and delete it
    Press Ctrl+A (or Cmd+A) to select all cells, then press Delete to clear them. This clears the old EAV data so the new import goes in cleanly.
  4. Import the matching CSV file
    Go to File → Import → Upload → select the matching CSV file (e.g. GSheet_01_Org.csv for the Org tab). In the import dialog:
    • Import location: "Replace current sheet"
    • Separator type: Comma
    • Convert text to numbers/dates: Yes
    Then click Import data.
  5. Verify the data landed correctly
    Scroll through the sheet. Row 1 should be the column headers. Rows 2+ should be the data. Make sure values with commas (like addresses) are in a single cell — not split across columns.
  6. Repeat for all 4 tabs
    Do Programs → SubPrograms → Events in the same order. The website reads all tabs on every sync, so all 4 must be updated for the site to reflect the full dataset.
  7. Wait 30 seconds, then reload the website
    The sheets.js engine polls every 30 seconds. After the import is done, reload any page on the website and the new content will appear automatically.

6How to Test That It's Working
  1. Open the website homepage
    Open Homepage.html and open your browser's developer console (F12). You should see [H.E.S.] log messages as the sheet syncs.
  2. Check the stats bar
    The three impact stats (100+ years, 5,000+ families, 20+ programs) should appear from the Org tab. If they're blank, the Org tab import didn't land correctly.
  3. Open a program page
    Open Summer Camp.html. The sidebar should show sub-program tabs (Day Camp, Teen Unit, Camp Xtra, Vacation Program). If you see only "Overview" and "Register", the SubPrograms import is missing or the entity name doesn't match exactly.
  4. Check events
    Open Community Events.html → "Upcoming Events" tab. All 6 events should appear. If the list is empty, the Events tab import didn't work.
  5. Make a live edit to test sync
    Change one value in the Google Sheet (e.g. the phone number in the Org tab). Wait 30 seconds. Reload the website. The new value should appear in the header or footer without any code change.
